## 功能
1. 點擊button可以切換頁面
## 開始使用
1. 安裝node.js與npm
2. 將專案clone到本地
3. 在本地開啟後，在終端機進入資料夾，輸入：
```bash
npm install
```
4. 安裝完畢後輸入：
```bash
npm run start
```
5. 若看見此行訊息代表順利運行，打開瀏覽器輸入以下進入網址
```bash
localhost:3000
```
6. 若要暫停使用
```bash
ctrl + c
```

## 開發工具
- Node.js 10.15.0
- Express 4.17.1
- Express-Handlebars 5.3.3
- Bootstrap 5.2.1
